# 💻 Cisco Networks – Prácticas de Networking

## Este repositorio documenta mi aprendizaje práctico en el curso Conceptos Básicos de Redes de Cisco Networking Academy.

## 📂 Contenido del repositorio

| Módulo                                             | Descripción                                                       |
| -------------------------------------------------- | ----------------------------------------------------------------- |
| **Módulo 4 – Redes domésticas**                    | Configuración inicial de routers y clientes inalámbricos.         |
| **Módulo 8 – Conectarse a servidores**             | Prácticas para conectarse a un servidor web usando Packet Tracer. |
| **Módulo 11 – DHCP**                               | Configuración de asignación dinámica de IP en routers.            |
| **Módulo 12 – NAT y puertas de enlace**            | Configuración y observación de NAT y gateways en red doméstica.   |
| **Módulo 13 – Identificación de IP y MAC**         | Reconocimiento de direcciones IP y MAC en la red.                 |
| **Módulo 14 – Creación de LAN y flujo de tráfico** | Observación del flujo de tráfico y creación de LAN.               |
| **Módulo 16 – Aplicaciones y servicios de red**    | Interacción cliente-servidor, FTP, Telnet, SSH y solicitudes web. |
| **Módulo 17 – Comandos de diagnóstico**            | Uso de `ping` e `ipconfig` para pruebas de conectividad.          |


---

## 📝 Cada carpeta de módulo contiene:
- Archivos de laboratorio (`.pka`)  
- Capturas de pantalla de prácticas  
- Notas y resúmenes del módulo
